JavaScript 简介

JavaScript 能做什么?

JavaScript 能够改变 HTML 内容。

JavaScript 能够改变 HTML 属性值。

在本例中，JavaScript 改变了图像的 src 属性值。

JavaScript 能够改变 HTML 元素的样式。

JavaScript 能够隐藏和显示 HTML 元素。

JavaScript 输出

JavaScript 不提供任何内建的打印或显示函数。

JavaScript 显示方案

JavaScript 能够以不同方式“显示”数据：(5+6)

  • 使用 window.alert() 写入 警告框

    这会显示一个警告框

  • 使用 document.write() 写入 HTML 输出

    只能在 HTML 输出流中使用

  • 使用 innerHTML 写入 HTML 元素

  • 使用 console.log() 写入 浏览器控制台

    通过 F12 来激活并选择浏览器控制台
